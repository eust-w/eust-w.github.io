## 流程
1. 安装avahi服务

2. 安装avahi-tools

   ```
   sudo yum install avahi-tools
   ```

3. 重启dbus

4. 修改 vi /etc/avahi/services/zstackAPT.service

```sh
<?xml version="1.0" standalone='no'?>
<!DOCTYPE service-group SYSTEM "avahi-service.dtd">
<service-group>
    <name>zstack@*APT</name>
    <service>
        <type>_zstack@*APT._tcp</type>
        <port>7654</port>
    </service>
</service-group>
```
